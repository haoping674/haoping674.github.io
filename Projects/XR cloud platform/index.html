關於專案

通過 ASP.NET Core 建立具有報名元宇宙職災訓練課程的平台。可供管理員對職災訓練的細節進行調整，並且可實時觀看報名人數。使用者可以進行團隊報名或是個人報名。

使用工具

  • 前端：HTML、Bootstrap、Razor、Ajax
  • 後端：C# ( ASP.NET )
  • 資料庫：MySQL
  • 開發工具：Visual Studio、Github、Git

特點

  • 4 種身分機制：普通使用者、組織管理員、課程管理員、系統管理員
  • 使用 GUID 創建使用者識別碼
  • 組織管理員可批量報名課程
